# AIMMD-TIS-Framework

This repository contains the code and resources for the AIMMD-TIS method from the paper: "Understanding Reaction Mechanisms from Start to Finish".
  
## What this repository includes

- **AIMMD-TIS integration** for running Transition Path Sampling (TPS) and Transition Interface Sampling (TIS) workflows with OPS.
- **Example setups** to reproduce key steps from the paper.
- **Utilities and analysis scripts** for trajectory handling and mechanism interpretation.

## Quick start

1. Install dependencies (AIMMD, OpenPathSampling, and project requirements).
2. Prepare an input system and simulation settings.
3. Run TPS/TIS workflows using the provided scripts.
4. Analyze generated trajectories with the included tools.
